# React + Vite 개인포폴
- 개인 포트폴리오를 위한 사이트 재구축 
- 대상 사이트는 국내 스포츠 판매 사이트 '프리모상사'를 선정
  - 주소 : "http://www.primosports.co.kr/"
- 단, 타 쇼핑몰 사이트 형식으로 재구축 하지 않음.
- 1차 Front 완성 후 2차 Back단 작업 예정
